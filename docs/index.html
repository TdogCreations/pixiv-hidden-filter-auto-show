Always show works hidden by Pixiv's search filter

A free Chrome & Firefox extension that permanently turns off Pixiv's "Works that may not be relevant to your search" filter — the one Pixiv resets on every single search.

Download for Chrome / Edge / Brave Download for Firefox Source on GitHub

Chrome Web Store & Firefox Add-ons listings are pending review — for now, install from the GitHub release (instructions in the repo).

Does this solve your problem?

  • Pixiv shows "Works that may not be relevant to your search" and hides results you actually want to see.
  • You set Search options → Other → "Works that may not be relevant to your search" → Show, but it resets to "Hide" on the very next search.
  • Pixiv is suddenly showing fewer search results.
  • You want to show all Pixiv search results and disable the relevance filter permanently.

If any of those sound familiar, install it and forget about it — the filter stays off automatically on every search.

Features

  • Always on. Adds dgw=1 to every Pixiv search and tag URL so hidden works always appear.
  • Survives navigation. The filter stays off when you click tags, paginate, change sort order, or hit back/forward.
  • Optional logo redirect. Make the Pixiv logo open your followers' new-works feed instead of the recommendations page.
  • Private. No data collected, no external servers, no remote code. Just one local on/off setting.
  • English & Japanese. The popup follows your browser language automatically.
  • Lightweight. Plain JavaScript, Manifest V3, no trackers, no dependencies.

Pixiv は検索のたびに 「検索を妨げる可能性がある作品」 フィルターをリセットします。検索オプション → その他の条件 で 「表示する」に設定しても、検索するたびに「表示しない」に戻って しまいます。

Privacy

The extension does not collect, transmit, log, or share any personal data. It stores only the on/off toggle settings using your browser's built-in sync storage, and contacts no external server. Full policy: PRIVACY.md.
